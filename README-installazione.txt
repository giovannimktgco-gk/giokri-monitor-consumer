INSTALLAZIONE GITHUB PAGES

1. Crea un nuovo repository GitHub chiamato:
   giokrimonitorconsumer

2. Carica tutti i file contenuti in questa cartella.

3. Vai in:
   Settings -> Pages

4. Seleziona:
   Branch: main
   Folder: /root

5. Salva.

6. Dopo alcuni minuti il sito sarà online.

IMPORTANTE:
Prima di usare l'app, crea la tabella SQL in Supabase usando lo script fornito da ChatGPT.
